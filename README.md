# 📦 Análise de Estoque com Python

Projeto desenvolvido para automatizar a análise de estoque, com foco em identificação de ruptura, excesso e baixa cobertura.

---

## 🎯 Objetivo

Analisar dados de estoque e gerar alertas que apoiem a tomada de decisão logística.

---

## ⚙️ Funcionalidades

- Identificação de produtos em ruptura
- Detecção de excesso de estoque
- Classificação de prioridades
- Geração de relatórios automáticos

---

## 🛠️ Tecnologias

- Python
- Pandas
- Análise de dados
