# 📦 Automação e Análise de Estoque com Python

Projeto desenvolvido para automatizar a análise de estoque, com foco em identificação de ruptura, excesso e baixa cobertura, apoiando a tomada de decisão operacional em logística.

---

## 🎯 Objetivo

Automatizar a leitura de dados de estoque e gerar indicadores que permitam:

- Identificar produtos em risco de ruptura
- Detectar excesso de estoque
- Classificar prioridades de ação
- Apoiar decisões de reposição e planejamento

---

## ⚙️ Funcionalidades

- 🔴 Identificação automática de produtos em ruptura  
- 🟡 Detecção de baixa cobertura de estoque  
- 🟢 Identificação de excesso de estoque  
- ⚠️ Classificação de itens com prioridade crítica  
- 📊 Geração de relatório analítico automático  

---

## 🛠️ Tecnologias utilizadas

- Python 3  
- Pandas  
- Manipulação de dados (CSV / Excel)  
- Lógica de negócio aplicada à logística  

---
